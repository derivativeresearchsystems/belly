# g

> A simplified Git CLI.


## Install

```
$ npm install --global g
```


## Usage

```
$ g --help

  Usage
    g [input]

  Options
    --foo  Lorem ipsum [Default: false]

  Examples
    $ g
    unicorns & rainbows
    $ g ponies
    ponies & rainbows
```
